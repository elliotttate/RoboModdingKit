# HelloRoboLogMod

This is the smallest useful UE4SS sample mod in the kit.

What it does:

- loads as a normal UE4SS Lua mod
- prints a marker line into `UE4SS.log`

Use it to verify that:

- the UE4SS runtime is deployed correctly
- `install_mod.ps1` can install a loose UE4SS mod directory
- `package_mod.ps1` can create a distributable `.zip`
