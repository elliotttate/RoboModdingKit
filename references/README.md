# References

This folder is for local reference outputs generated from your own RoboQuest install.

Typical subfolders:

- `references/dumps`
- `references/kismet_json`
- `references/pseudocode`
- `references/sdk_generated`
- `references/sdk_dump_tools`
- `references/ida`

The public repo intentionally does not commit those game-derived artifacts.

Relevant docs:

- `docs/05_dumps_sdk_assets.md`
- `docs/10_generation_pipeline.md`
